# Installation

1. Download the latest AIT Multilingual Translate release ZIP.
2. In WordPress, open **Plugins > Add New > Upload Plugin**.
3. Upload the release ZIP.
4. Activate **AIT Multilingual Translate**.
5. Open the plugin admin menu in WordPress.
6. Choose the source language and target languages.
7. Add provider API keys.
8. Run the Scanner.
9. Process the Translation Queue.

Always test on staging before production usage.
